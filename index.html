Example Work
Project 1: Team Project Web APIs

A team project designed to practice the "fetching" of 3rd party web APIs. Our team created The Dad DOJO. Using HTML, CSS, Javascript and several other third party APIs we created a website that allows users to search new BBQ recipes, "Dad" jokes, and cocktail recipes. Each modal lets the user cycle through a variety BBQ recipes, cocktail recipes, and jokes. If a user likes a joke or recipe our site allows them to save it to their "favorites" section and it is stored in local storage for the next time they visit the site.
